## Zsh

1. install zsh packages
  
  `sudo pacman -S zsh zsh-syntax-highlighting zsh-autosuggestions`

2. create directories

  `mkdir $HOME/.local/state/zsh`

  `mkdir $HOME/.cache/zsh`
